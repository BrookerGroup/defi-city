# Asset Pack Files

Download the isometric asset pack from:
https://sweeetpotatoo.itch.io/isometric-buildings-and-tiles

Place the following files in this directory:

Required files:
- `buildings-1.png` - Building sprites (128x128 each, 4 buildings in a row)
- `grass-tiles.png` - Grass tile sprites (128x64 each)

Optional files (for future features):
- `buildings-2.png` - Additional building sprites
- `water-tiles.png` - Water tiles
- `street-tiles-1.png` - Road tiles
- `street-tiles-2.png` - Additional road tiles
- `floor-walls.png` - Wall and floor tiles

## Sprite Sheet Layout

### buildings-1.png
The sprite sheet contains 4 buildings in a horizontal row:
- Position 0 (0-127px): Town Hall
- Position 1 (128-255px): Bank
- Position 2 (256-383px): Shop
- Position 3 (384-511px): Lottery

Each building sprite is 128x128 pixels.

### grass-tiles.png
Isometric grass tiles at 128x64 pixels each.
